类型系统基础速查表 Stage 1 Cheat Sheet

阶段一(0001–0007)的压缩精华,随时回查。权威来源: 官方 Handbook。

基本类型 & 注解

let s: string = "a";
let n: number = 1;
let b: boolean = true;
function f(x: number): string { return ""; }

推断 & 何时省略

位置做法
初始化变量省略,靠推断
函数参数必写
对外 API 返回建议显式

字面量 & 联合 & 别名

type Dir = "up" | "down";
let id: number | string;
const c = { x: 1 } as const; // 锁字面量

收窄 Narrowing

typeof x === "string"
"prop" in obj
x instanceof Cls
// 可辨识联合: 用 kind 字段 + switch

对象类型

{ name: string;      // 必填
  nick?: string;     // 可选
  readonly id: number; // 只读
  [k: string]: unknown; // 索引签名
}

数组 / 元组 / 枚举

number[]              // 数组
readonly number[]     // 只读数组
[string, number]     // 元组
type R = "r"|"g"|"b"; // 常优于 enum

函数类型

type Fn = (a: string) => void;
(a: number, b?: number) => number
(...xs: number[]) => number // rest

特殊类型 & 断言

any     // 关检查,避免
unknown // 安全 any,用前收窄
never   // 不可能/穷尽检查
x as string   // 断言(非转换)
y!            // 非空断言
