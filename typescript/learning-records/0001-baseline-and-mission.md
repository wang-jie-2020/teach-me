# 建立基线：有一些 JS 基础，目标为职业技能

第一次会话确立了 mission 与起点，用于计算后续的最近发展区。

- **背景（用户自述）**：有“一些 JavaScript”，但不完全自信；方向（前端/后端）未定。
- **含义**：讲解可假设用户认识变量、函数、对象等 JS 概念，但需在关键处补齐（如 `typeof`、`undefined`、`===`）。避免假设精通闭包、异步、原型链等。
- **偏好**：动手写代码 + 清晰讲解 + 速查表 + 真实项目（四者兼顾）；中文教学；每课要短、可完成、即时反馈。
- **教学方向**：先教“到处都用得上”的 TS 核心（类型注解、推断、对象/union、interface/type），暂不绑定框架，直到方向明确。

第 0001 课覆盖了“TS 是什么 + 日常基本类型 + 推断直觉”。**注意**：这是“已覆盖”，尚非“已掌握”——需等用户完成练习并展示正确使用后，才据此写新记录并向 GLOSSARY 提升术语。
