# TypeScript Resources

高信任来源优先。知识主要来自官方 Handbook；社区用于获取“智慧”（真实反馈）。

## Knowledge

- [官方文档：The TypeScript Handbook](https://www.typescriptlang.org/docs/handbook/intro.html)
  官方权威指南，面向日常开发者。首选阅读来源。Use for：任何语言特性的准确定义与行为。
- [官方文档：Everyday Types](https://www.typescriptlang.org/docs/handbook/2/everyday-types.html)
  最常用类型（string/number/boolean、数组、对象、union、可选属性等）的权威讲解。Use for：第 1~2 课的基础类型。
- [官方文档：TS for the New Programmer](https://www.typescriptlang.org/docs/handbook/typescript-from-scratch.html)
  用最少假设解释 TS 是什么、为何存在。Use for：理解 TS 相比 JS 的价值（面试常问）。
- [官方交互环境：TypeScript Playground](https://www.typescriptlang.org/play)
  浏览器里即时编译、看类型推断与报错，无需本地安装。Use for：每课的动手练习与验证。
- [官方 Cheat Sheets](https://www.typescriptlang.org/cheatsheets)
  官方出品的语法速查图（类型、接口、类、控制流）。Use for：制作/校对本工作区 reference 文档。
- [Handbook · Narrowing](https://www.typescriptlang.org/docs/handbook/2/narrowing.html)
  类型收窄权威页。Use for：0003 收窄、可辨识联合、穷尽检查。
- [Handbook · Object Types](https://www.typescriptlang.org/docs/handbook/2/objects.html)
  对象类型、可选/只读/索引签名。Use for：0004 对象类型。
- [Handbook · More on Functions](https://www.typescriptlang.org/docs/handbook/2/functions.html)
  函数类型、重载、void。Use for：0006 函数类型。

## Wisdom (Communities)

- [r/typescript](https://www.reddit.com/r/typescript)
  官方 TS 子版块，讨论质量较高。Use for：代码点评、实践困惑、求职方向讨论。
- [Stack Overflow — typescript 标签](https://stackoverflow.com/questions/tagged/typescript)
  海量高质量问答。Use for：调试具体报错、查“为什么类型不匹配”。
