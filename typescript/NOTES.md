# Notes

## 用户偏好
- 学习方式：动手写代码 + 清晰讲解 + 可回顾速查表 + 真实项目（四者都要）。
- 语言：中文教学（课程 HTML、文档均用中文，代码与关键术语保留英文原词）。
- 每节课要短、可快速完成，给出即时反馈。

## 背景
- 目标：把 TS 作为职业技能（求职/工作），方向（前端/后端）暂未确定。
- 当前水平：有“一些 JavaScript”，但不完全自信 —— 需要时补齐 JS 缺口，不假设精通。

## 教学策略
- 优先教“到处都用得上”的 TS 核心，暂不绑定框架。
- 每课一个可验证的小胜利（能写出 / 能读懂 / 能修复某类报错）。
- 术语首次出现给中英对照；沿用 GLOSSARY 的统一译法。
- 工作区约定：每个学习主题一个子文件夹（本主题在 typescript/）。
